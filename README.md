# Employee-database-in-a-file-with-C
## Description
**A Command line program for managing an employee data set.**

**Each employee has:**
- First name
- Last name
- Age
- Address
- Salary
## How to use:
The script compile.sh allows to compile the whole project into an executable file in linux named main.out
Execute script.sh: 
  ./script.sh
Execute main.exe:
  ./main.exe
